# Sportsee 🏃‍♂️📊

## 📌 Contexte

Sportsee est un tableau de bord sportif développé dans le cadre de ma formation **OpenClassrooms**. Le projet consiste à créer une application React affichant les données d'activité physique d'un utilisateur (session, distance, fréquence cardiaque, objectifs) sous forme de graphiques interactifs, en s'appuyant sur une API backend fournie.

## ✨ Fonctionnalités

- Authentification (login / logout) avec routes protégées
- Graphique de distance hebdomadaire avec pagination par semaines
- Graphique de fréquence cardiaque (min/max + moyenne) avec navigation semaine par semaine
- Donut chart de suivi des objectifs de course hebdomadaires
- Calcul de métriques dérivées (total de calories, jours de repos)

## 📦 Dépendances principales

- **React** + **React Router v8** (mode framework / SSR)
- **TypeScript**
- **Recharts** (graphiques)
- **Vite** (build)
- **Node.js / Express** (backend fourni)

## 🚀 Installation

```bash
git clone https://github.com/<ton-utilisateur>/sportsee.git
cd sportsee
npm install
npm run dev
```
